# 📘 English Janala

## 📝 Description
English Janala is a simple and interactive vocabulary learning web application where users can explore English words by lessons, search for specific words, and listen to their pronunciation. The application dynamically fetches data from external APIs and presents it in a clean, responsive UI.

It follows a data-driven approach using JavaScript fetch() to load lessons and word details in real-time, making the app scalable and easy to maintain.

---

## 🚀 Features
- 🔎 Dynamic word search using API  
- 📚 Lesson-based vocabulary browsing  
- 🔊 Word pronunciation with Text-to-Speech  
- 📖 Detailed word view with modal  
- ⚡ Real-time data fetching
